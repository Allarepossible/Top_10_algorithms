Статья с проведенными исследованиями по основным сортировкам:

![alt text](https://habrastorage.org/getpro/habr/post_images/b91/1bc/ca9/b911bcca9ca9f9d8b0fa781a49118553.png)

Практически во всех статьях побеждает "быстрая" сортировка ("quick" sort). Несмотря на то, что другие сортировки
быстрее обрабатывают небольшие массивы, этот выигрыш во времени совсем незначителен. В свою очередь на больших объемах
данных быстрая сортировка начинает побеждать и потребляет намного меньше памяти.
